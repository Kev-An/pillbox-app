# Pillbox

Cross-platform medication reminder application built with React Native and Expo.

## Features

- Medication scheduling
- Push notifications (FCM)
- Medication histor.jpegy tracking
- JWT Authentication
- MongoDB backend

## Tech Stack

- React Native
- Expo
- Node.js
- MongoDB
- Firebase Cloud Messaging
